Description

Creates a address book table and add shipping address of the logged in user.

This simple extension allows you to add shipping address to addressbook table.



Installation 

1. Unzip into your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
